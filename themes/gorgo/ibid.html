IBid Mobile

Summer 2020 ||| Technologies Used: Technologies Used: VueJS, HTML, CSS, Javascript, MySQL, JQuery ||| Front-End Repository ||| Live Version

This summer, I personally built a Full-Stack application which allows users to log in, view , save and bid on items. Each event dynamically styles its items and its event information using a VueJS front-end server, while connecting to a mySQL database using ASP.NET routes.

The styling is designed to replace an existing site. On the live version, login, favoriting, and event switching not be permitted due to lack of database connection.
